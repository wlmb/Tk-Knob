Tk::Knob

This archive contains the distribution Tk::Knob, version 0.001 to build knob widgets that can rotate continuously and indefinitely in any direction under Tk.

Read the file INSTALL for installation instructions.

Read lib/Tk/Knob.pm for usage instructions and other information.
